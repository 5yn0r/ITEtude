# Fiche Projet : ITEtude V2.0

## 📝 Présentation Générale
**ITEtude** est une solution de curation intelligente conçue pour la communauté informatique francophone. Elle transforme le flux massif d'informations en ligne en parcours d'apprentissage structurés et cohérents.

## 🎯 Objectifs
- **Guider** les apprenants dans l'océan de ressources IT.
- **Structurer** l'apprentissage par des parcours thématiques.
- **Valider** les acquis via le suivi de progression et les certifications.
- **Fédérer** une communauté d'entraide francophone.

## ✨ Fonctionnalités Clés
### 🤖 Intelligence Artificielle (Genkit)
- **Recherche IA** : Un assistant pédagogique qui analyse les requêtes en langage naturel et recommande les meilleures ressources avec une justification personnalisée.
- **Vérification de Liens** : Système automatique détectant les liens brisés pour garantir une base de données toujours opérationnelle.

### 📚 Apprentissage & Contenu
- **Parcours Structurés** : Chemins balisés du niveau débutant à avancé.
- **Gestion des Ressources** : Filtrage par difficulté, langue et "poids data" (éco-conception).
- **Favoris & Notes** : Espace personnel pour marquer des ressources et prendre des notes d'étude.

### 👥 Communauté
- **Salon Principal** : Chat en temps réel pour échanger avec d'autres apprenants.
- **Bandeau WhatsApp** : Intégration directe pour rejoindre le groupe de discussion instantanée.

### 🛡️ Administration & Sécurité
- **Panneau Admin (Secret)** : Accessible via `/postadeep` pour la gestion totale des données.
- **Rôles Firebase** : Sécurité stricte via Firestore Rules et authentification multi-facteurs.

## 🛠️ Stack Technique
- **Frontend** : Next.js 15 (App Router), React 19, Tailwind CSS.
- **UI Components** : Shadcn/ui, Lucide React.
- **Backend** : Firebase (Auth, Firestore).
- **IA Framework** : Genkit v1.x (Google Gemini 2.5 Flash).
- **Type Safety** : TypeScript, Zod.

## 🚀 Vision Future
Devenir la plateforme de référence pour l'auto-formation IT en Afrique et dans tout l'espace francophone, en intégrant davantage de gamification et de mentorat.
